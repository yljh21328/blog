[vim] 安裝顏色模板 color scheme

by Chris Yang
in vim on 2014-07-01

預設的 vim 顏色很難看

利用 color scheme 可以讓 vim 長的好看一些

可以在 Vim Colorscheme Gallery 找一個適合自己的模板

我是用 jellybeans

安裝方式如下

下載 jellybeans.vim
mkdir ~/.vim/colors
mv jellybeans.vim ~/.vim/colors

然後設定.vimrc

vim ~/.vimrc

colorscheme jellybeans
set t_Co=256
syntax on
