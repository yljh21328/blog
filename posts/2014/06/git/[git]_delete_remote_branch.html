[git] 刪除遠端的branch

by Chris Yang
in git on 2014-06-19

要刪除本地端的 branch 非常容易

先切換到其他的branch

然後下 git branch -d branch-name 即可

然而如果要刪除遠端的branch

則必須下 git push branch --delete branch-name
